Hues of Blue

Collaborated by myself and three other developers to build web application connecting those who need help dealing with psychological issues to mentors willing to provide it. Users can register and request connections. Upon connection, communication between the users will be enabled.
Technologies used: Python, Pylot, MySQL, Apache, HTML5, CSS3, Twitter Bootstrap

Warning Signs

This quiz app was developed by a group of 6 developers at AtHackathon to End Human Trafficking in San Francisco in April, 2016. The user is guided through yes or no questions about a neighbor they are concerned for. At the end the quiz, the user is given an assessment and linked to relevant resources to help. Technologies used: JavaScript, Angular.js, MongoDB, Node.js, Express.js, Bootstrap, CSS3, HTML5
